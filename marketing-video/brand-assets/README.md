# Hub Suite Brand Assets

This folder stores the reusable logo and color system that worked from the promo direction.

It intentionally does not include the rough promo video scenes, background boxes, screenshot crops, or any central "Your Company" / "The Company" hub mark.

## Product Logos

- `logos/service-desk.svg` uses `#246BFD`
- `logos/human-hub.svg` uses `#00A676`
- `logos/project-hub.svg` uses `#D99A21`
- `logos/customer-hub.svg` uses `#C14D7A`
- `logos/insight-hub.svg` uses `#5D5FEF`

## Usage Notes

Use the logo SVGs as standalone product marks on clean white or light neutral surfaces. If these are used in future video work, avoid placing large product cards behind screenshots unless the layout has been designed frame-by-frame.
